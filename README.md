# ETHEREUM

Brief descriptions of the repo content:

- `0.6.1.SharedWallet/`: Shared wallet contract example and deployment notes.
- `0.6.12.ERC777PresetFixedSupply/`: ERC777 preset fixed-supply token (Solidity 0.6.12) plus reference README and single-file build.
- `0.6.20.ERC777PresetFixedSupply.sol`: ERC777 preset fixed-supply token (Solidity 0.6.20).
- `0.7.0.ERC777PresetFixedSupply.sol`: ERC777 preset fixed-supply token (Solidity 0.7.0).
- `0.7.0.GameOfSecrets.sol`: Simple commit-reveal style “Game of Secrets” contract (Solidity 0.7.0).
- `0.8.2.Notary.sol`: Simple on-chain notary / secret hash registry (Solidity 0.8.2).
- `Ethereum-tx-contracts/`: ERC20/ERC1155 token contracts, upgradeable variants, and published contract artifacts.
